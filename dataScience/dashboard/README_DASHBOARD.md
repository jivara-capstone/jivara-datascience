# Dashboard Jivara - Panduan Pengguna

Dashboard ini adalah visualisasi interaktif dari hasil Data Science pipeline Jivara untuk drug-food interaction detection system.

##  Cara Menjalankan

### Prerequisite
- Python 3.9+
- Dependencies: `streamlit`, `pandas`, `plotly`, `scipy`

### Install Dependencies
```bash
pip install streamlit pandas plotly scipy
```

### Run Dashboard
```bash
cd d:\jivara\dataScience\dashboard
streamlit run dashboard.py
```

Dashboard akan buka di: `http://localhost:8501`
